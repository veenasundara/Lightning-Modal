# Lightning-Modal
Component for Modal

## Usage
<!-- if allowClose is set to false, the "X" to close the modal will not be present -->
<c:CmpModal title="Title For Modal" allowClose="true/false">

	<!-- place your action buttons in the footer of the modal -->
	<aura:set attribute="footer">
	</aura:set>

	<!--- body of your modal here -->
</c:CmpModal>
